Laboratoria

Laboratoria enseña a jóvenes mujeres a crear productos digitales con el proposito de insertarlas a la creciente economía digital del Perú

En un periodo de 6 meses,nuestras coders aprenden:

  1. El lenguaje del futuro

    La programación está de detras de todos los productos digitales que usamos día a día. Nuestras coders aprenden HTML5, CSS3 y Java script, terminando preparadas para crear infinitos productos y soluciones web

  2. Aprender a prender

    Para triunfar como desarrolladoras web nuestras coders se convieren en autodidactas, descubriendo que no hay limites para seguir creciendo en este campo
